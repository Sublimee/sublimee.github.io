Claude Code Best Practice

Boris Cherny — создатель Claude Code. Кто мы такие, чтобы не прислушаться к его советам, как собственно этот Claude Code использовать?

Для этого можно, конечно, начать отслеживать его X Twitter, где он часто делится своим опытом... Но, как не сложно догадаться, нашелся энтузиаст, который собрал для нас всё в одном месте: howborisusesclaudecode.com . Отстающая по актуальности данных версия на русском — тут.

Почитать стоит даже если вы не используете Claude Code (долой vendor lock!) по двум причинам:

  • часть советов имеют общий характер;
  • Claude Code можно научить работать с любой LLM с помощью LiteLLM (тут, конечно, надо сделать ремарку, что лучше всего клодкод работает с антропиковскими моделями).
